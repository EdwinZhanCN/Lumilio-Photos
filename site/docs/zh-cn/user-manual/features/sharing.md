# 分享

<!-- 本文为功能参考文档，面向普通用户。截图占位符为 `TODO: 截图`，后续补充。 -->

分享功能用于把一组资源、一个相册或一个人物的照片生成公开链接。接收方无需登录 Lumilio Photos，也不会进入你的完整资源库，只能访问该分享链接包含的内容。

## 可以分享什么

当前可以从这些入口创建分享链接：

| 入口 | 分享范围 |
|------|----------|
| 资源库、喜欢、相册、人物、智能分类等资源浏览页的选择模式 | 当前选中的资源快照 |
| 全屏查看器 | 当前正在查看的单个资源 |
| 相册详情页顶部的分享按钮 | 整个相册，由服务端解析资源范围 |
| 人物详情页顶部的分享按钮 | 该人物对应的照片集合，由服务端解析资源范围 |

“选中资源快照”表示链接创建时会记录当时选中的资源集合。之后你在资源库中继续筛选或改变选择，不会影响已经创建的链接。

## 创建分享链接

打开分享弹窗后，需要填写或选择：

- **标题** — 用于在公开分享页和管理列表中识别这条链接。
- **有效期** — 可选 7 天、30 天或 90 天。
- **允许下载** — 开启后，接收方可以下载分享内容。
- **包含原始文件** — 仅在允许下载后出现；开启后下载会包含原始文件。

单击「创建链接」后，系统会生成公开 URL。

::: warning 重要
请在创建成功后立即复制链接。出于安全设计，服务端只保存分享令牌的哈希值，不保存原始令牌；因此完整 URL 只会在创建成功弹窗中显示一次。关闭弹窗后，无法从管理页面重新复制同一条链接。
:::

## 接收方看到什么

接收方打开 `/s/<token>` 形式的链接后，会看到一个独立的公开分享页：

- 不需要登录。
- 不会显示应用侧边栏、导航栏或 AI 助手面板。
- 只显示分享标题、到期时间、资源数量和被分享的资源。
- 可点击资源进入公开全屏查看。
- 如果分享者允许下载，页面会显示下载入口。

如果链接已过期、已撤销或不存在，接收方会看到“此链接已失效”的提示。

## 管理分享链接

你可以在[工具 → 分享链接](/zh-cn/user-manual/features/utilities#分享链接)中管理已经创建的链接。

管理页按状态筛选：

- **有效** — 仍可访问。
- **已过期** — 已超过到期时间。
- **已撤销** — 已被手动撤销。

有效链接支持：

- **延长** — 将有效期延长 30 天。
- **撤销** — 立即让链接失效。

已过期或已撤销的链接支持：

- **删除** — 从管理列表中移除记录。

管理页会显示每条链接的来源类型、资源数量、创建时间、到期时间、最后查看时间和查看次数。

## 常见问题

### 我关闭了创建弹窗，还能再复制链接吗？

不能。完整链接只在创建成功时显示一次。若链接丢失，建议撤销原链接并重新创建。

### 撤销链接会删除资源吗？

不会。撤销只会让公开链接失效，不会删除资源库中的原始资源，也不会改变相册、人物或标签。

### 接收方需要账号吗？

不需要。公开分享页运行在独立路由下，不会要求接收方完成登录、注册或首次设置。

### 允许下载和包含原始文件有什么区别？

允许下载决定接收方是否能下载分享内容；包含原始文件决定下载包中是否包含原始媒体文件。未开启允许下载时，接收方只能在线查看。
