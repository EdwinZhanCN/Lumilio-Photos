# 合集

<!-- 本文为功能参考文档，面向普通用户。截图占位符为 `TODO: 截图`，后续补充。 -->

合集是资源库的概览主页，在一个页面内聚合了多种浏览和组织入口，包括相册、地点与旅行、人物识别、文件夹、标签以及工具。

## 地图与旅行

地图页面以交互式地图展示所有带有地理位置信息的资源。

页面入口位于合集中的「地点」板块，点击可进入全屏地图视图。

### 旅行

系统会根据资源的地理位置和拍摄时间自动将资源归为「旅行」分组——当你在一个城市范围内、时间跨度不超过数天的连续拍摄会被自动识别为一次旅行。

每次旅行的卡片显示目的地城市、国家、照片数量和一张代表性封面图。点击旅行卡片进入该旅行的资源浏览页，可查看限定于该时段和地点的所有资源。

## 人物

人物页面展示通过人物识别技术自动聚类出的所有人物。每个识别出的人物以头像缩略图 + 名称 + 照片数量的卡片形式展示。

- **浏览** — 单击人物卡片进入该人物的详情页，浏览包含该人物的所有照片。
- **重命名** — 在人物详情页中可以为人物设置名称。设置名称后该人物标记为已确认。
- **纠错** — 在人物详情页的编辑弹窗中，可以移动或移除错误归属的人脸，也可以合并重复人物。
- **分享** — 为整个人物照片集合创建公开分享链接，或只分享选中的几张照片。

::: tip 提示
人物识别功能需要在[设置 → AI](/zh-cn/user-manual/features/settings#ai) 中开启人物识别 ML 功能。
:::

详情见[人物](/zh-cn/user-manual/features/people)。

## 文件夹

文件夹入口按仓库中的目录结构浏览资源。它不是相册，也不提供重命名或移动目录的编辑能力；文件夹页只根据资源所在路径生成浏览视图。

进入某个文件夹后，可以递归浏览该文件夹及其子文件夹中的资源，并通过顶部的子文件夹按钮继续下钻。详情见[工具 → 文件夹](/zh-cn/user-manual/features/utilities#文件夹)。

## 标签

标签入口按手动标签和 AI 标签汇总资源。你可以搜索标签，也可以只查看手动标签或 AI 标签。点击标签后进入该标签对应的资源视图。

标签页与资源库中的标签筛选不同：标签页从标签集合出发进入资源视图，资源库筛选则是在当前资源列表内缩小结果。详情见[工具 → 标签](/zh-cn/user-manual/features/utilities#标签)。

## 工具

工具入口包含喜欢、重复照片、回收站、标签、分享链接和智能分类等实用视图。详见[工具](/zh-cn/user-manual/features/utilities)。
