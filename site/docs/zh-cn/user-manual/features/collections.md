# 合集

合集是资源库的概览主页，在一个页面内聚合了多种浏览和组织入口，包括相册、地图与旅行、人物识别以及实用工具。

## 地图与旅行

地图页面以交互式地图展示所有带有地理位置信息的资源。

页面入口位于合集中的「地点」板块，点击可进入全屏地图视图。

### 旅行

系统会根据资源的地理位置和拍摄时间自动将资源归为「旅行」分组——当你在一个城市范围内、时间跨度不超过数天的连续拍摄会被自动识别为一次旅行。

每次旅行的卡片显示目的地城市、国家、照片数量和一张代表性封面图。点击旅行卡片进入该旅行的资源浏览页，可查看限定于该时段和地点的所有资源。

## 人物

人物页面展示通过人物识别技术自动聚类出的所有人物。每个识别出的人物以头像缩略图 + 名称 + 照片数量的卡片形式展示。

- **浏览** — 单击人物卡片进入该人物的详情页，浏览包含该人物的所有照片。
- **重命名** — 在人物详情页中可以为人物设置名称。设置名称后该人物标记为已确认。
- **重建聚类** — 人物列表页顶部提供「重建」按钮，可对所有人脸嵌入向量重新运行聚类分析，适用于新增照片后更新识别结果。已命名的人物在重建后会保留原有名称。

::: tip 提示
人物识别功能需要在[设置 → AI](/zh-cn/user-manual/features/settings#ai) 中开启人物识别 ML 功能。
:::
