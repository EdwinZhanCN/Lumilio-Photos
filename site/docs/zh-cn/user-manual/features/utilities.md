# 工具

<!-- 本文为功能参考文档，面向普通用户。截图占位符为 `TODO: 截图`，后续补充。 -->

工具位于[合集](/zh-cn/user-manual/features/collections)页面中，用来进入一些按条件生成的资源视图和维护型页面。它们不是普通相册：大多数工具不会创建新的资源分组实体，而是基于资源库已有状态、标签、文件路径或检测结果展示一组资源。

## 工具入口

当前工具入口包括：

| 工具 | 用途 |
|------|------|
| **喜欢** | 查看所有已标记喜欢的资源，并可批量从喜欢中移除 |
| **重复照片** | 审查重复或高度相似的资源 |
| **回收站** | 浏览和恢复已移入回收站的资源 |
| **标签** | 按手动标签或 AI 标签浏览资源 |
| **分享链接** | 管理已创建的公开分享链接 |
| **文档 / 收据 / 插画** | 浏览 AI 自动分类出的智能分类资源 |

## 喜欢

喜欢页面展示所有已标记喜欢的资源。它使用与[资源库](/zh-cn/user-manual/features/assets)相同的浏览组件，因此支持布局切换、搜索、筛选、排序、选择模式和全屏查看器。

在喜欢页面中，批量操作会针对这个固定范围做简化：

- **分享** — 为选中的资源创建分享链接。
- **从喜欢中移除** — 将选中资源的喜欢状态取消。资源不会被删除，也不会从资源库中移除。

::: tip 提示
因为页面本身只显示已喜欢资源，所以这里不会再显示“标记为喜欢”的批量操作。
:::

## 重复照片

重复照片工具用于查找和管理资源库中的重复或高度相似照片。

### 扫描

在[管理](/zh-cn/user-manual/features/manage)页面对某个仓库执行「扫描重复文件」操作后，结果会出现在此页面。扫描基于两种方式：

- **精确匹配** — 文件内容完全一致的资源。
- **感知哈希** — 内容高度相似（如不同尺寸、不同压缩率的同一张照片）。

### 审查与处理

重复照片以分组形式展示。每组显示所有重复的缩略图，并推荐一个最佳保留项。你可以：

- **保留指定项** — 手动选择一组中要保留的资源。
- **合并** — 按推荐或手动选择保留项，处理同组中的其他重复资源。

## 回收站

回收站展示所有已删除的资源。从[资源库](/zh-cn/user-manual/features/assets)、相册详情或其他资源浏览页执行删除后，资源会移入此处。

- **浏览** — 回收站使用与资源库相同的浏览组件，支持搜索、筛选和全屏查看。
- **恢复** — 选中资源后，通过批量操作中的「恢复」将资源移回正常资源库视图。
- **批量操作限制** — 回收站中仅提供恢复相关操作，不提供评分、标记喜欢、堆叠、下载等普通资源操作。

::: warning 注意
当前普通删除是“移入回收站”。回收站中的恢复会影响资源在库中的可见状态，但不会修改原始媒体文件内容。
:::

## 标签

标签页面按标签汇总资源。它与资源库中的“标签筛选”不同：

- **资源库标签筛选**是在当前资源列表内缩小结果。
- **标签页面**是从所有标签出发，先选择一个标签，再进入该标签对应的资源视图。

标签列表支持搜索，并可按来源筛选：

| 来源 | 说明 |
|------|------|
| **全部** | 同时显示手动标签和 AI 标签 |
| **手动** | 仅显示用户手动添加的标签 |
| **AI** | 仅显示由 AI 自动分类或识别得到的标签 |

点击某个标签后，会进入该标签的资源页面。资源页面同样使用资源库浏览组件，可继续搜索、筛选、选择和全屏查看。

::: info 说明
同名标签会按来源区分。例如手动添加的“收据”和 AI 自动生成的“收据”可以作为不同来源的标签视图存在。
:::

## 文件夹

文件夹页面从资源所在的仓库路径中派生出来。它展示的是你在仓库中已有的目录结构，不是可编辑的相册。

文件夹列表按仓库分组显示。点击文件夹后进入详情页：

- 页面顶部显示文件夹名、所属仓库、资源数量和日期范围。
- 如果存在子文件夹，顶部会显示可横向滚动的子文件夹按钮。
- 下方资源列表递归显示该文件夹及其子文件夹内的资源。

::: tip 提示
文件夹用于按磁盘目录浏览资源。重命名、移动或管理目录本身应在仓库文件系统中完成；Lumilio Photos 的文件夹页只负责浏览。
:::

## 分享链接

分享链接页面用于管理你已经创建的公开分享链接。这里可以查看链接标题、来源、资源数量、创建时间、到期时间、最后查看时间和查看次数。

分享链接按状态分为：

- **有效** — 仍可访问的链接。
- **已过期** — 超过到期时间的链接。
- **已撤销** — 被你手动撤销的链接。

可执行的操作包括：

- **延长** — 将有效链接延长 30 天。
- **撤销** — 让有效链接立即失效。
- **删除** — 删除已过期或已撤销的链接记录。

::: warning 注意
分享链接的完整 URL 只会在创建成功时显示一次。关闭创建弹窗后，管理页面不会再提供“复制链接”按钮。如果链接丢失，需要撤销旧链接并创建新链接。
:::

更多创建与接收方访问说明见[分享](/zh-cn/user-manual/features/sharing)。

## 智能分类

系统通过 AI 自动将资源按内容主题分类为三个智能分类：

- **文档** — 包含扫描件、截图、文字拍摄等文档类资源。
- **收据** — 包含购物小票、发票等收据类资源。
- **插画** — 包含插画、漫画、海报等非摄影类图像资源。

每个智能分类本质上是一个按 AI 标签筛选的虚拟相册，资源由系统自动归入。在智能分类页面中，你可以像浏览普通相册一样查看和操作资源，同样支持搜索、筛选和全屏查看器。

::: info 说明
智能分类依赖于 AI 模型对资源内容的分析。相关 ML 功能（如零样本分类）可在[设置 → AI](/zh-cn/user-manual/features/settings#ai) 中管理。
:::
