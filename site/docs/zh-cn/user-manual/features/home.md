## 概览

首页是登录后的默认落地页，为概览型仪表盘。页面包含一个切换标签栏，在「精选」和「统计」两个视图间切换，下方始终显示时空地图卡片。

## 精选

精选视图以响应式网格展示一组精选照片（最多 8 张），照片带 3D 卡片效果。精选照片来源于近期拍摄和评分较高的资源，由系统自动筛选。

加载过程中显示骨架占位格。如果加载失败，顶部会显示错误提示横幅。

## 统计

统计视图以 2 列网格展示四张分析卡片，数据范围受当前选中的[工作库](/zh-cn/user-manual/features/settings#服务器)影响。

### 常用焦距

展示使用频率最高的 5 个焦距，以进度条显示各焦距的占比。

### 拍摄时间分布

以四个环形进度指示器展示照片在一天中不同时段的分布：

- **黄金时段**（日出后 1 小时 / 日落前 1 小时）
- **蓝调时刻**（日出前 / 日落后）
- **日出**
- **日落**

### 常用相机与镜头组合

列出最常用的相机 + 镜头搭配及各自的使用占比。

### 拍摄活动热力图

以 GitHub 贡献图风格的热力图展示全年每天的拍摄活动量。顶部可选择查看不同年份的数据。

## 时空地图

首页底部始终展示时空地图卡片。地图基于 Leaflet，标注所有带地理位置信息的照片。

- **位置标记** — 每张有 GPS 坐标的照片在地图上以点标记。
- **地点计数** — 卡片顶部显示已识别的地点聚类数量。
- **点击标记** — 单击照片标记跳转到 [资源库](/zh-cn/user-manual/features/assets) 中对应的资源。

无地理位置数据的照片不会出现在地图上。
