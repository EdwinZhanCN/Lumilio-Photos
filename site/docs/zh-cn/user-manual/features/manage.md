## 概览

管理库页面是媒体仓库的管理中心，也是上传新文件的入口。你可以在此查看所有已连接的仓库、执行仓库维护操作，以及通过拖拽或文件选择器上传资源。

## 上传资源

页面上方为上传区域，支持拖拽文件或单击选择文件。上传前会校验文件类型是否在服务端允许的范围内，不合法的文件会被自动过滤并给出提示。

- **上传目标** — 显示当前上传的目标仓库名称和路径（默认为当前选中的[工作库](/zh-cn/user-manual/features/settings#服务器)）。
- **添加文件** — 打开文件选择器批量选取文件。
- **清除** — 清空已添加但尚未上传的文件列表。
- **上传** — 开始上传所选文件，上传过程中显示进度和转码状态。

上传区域下方可展开信息面板，查看服务端的文件块大小、并发限制等配置。

### 支持的文件格式

页面顶部可点击"支持格式"按钮，打开弹窗查看所有可接受的文件扩展名。共支持 47 种文件格式，按类别分组如下：

**照片**（10 种） — `jpg` `jpeg` `png` `webp` `gif` `bmp` `tiff` `tif` `heic` `heif`

**RAW 格式**（13 种） — `cr2` `cr3` `nef` `arw` `dng` `orf` `rw2` `pef` `raf` `mrw` `srw` `rwl` `x3f`

**视频**（14 种） — `mp4` `mov` `avi` `mkv` `webm` `flv` `wmv` `m4v` `3gp` `mpg` `mpeg` `m2ts` `mts` `ogv`

**音频**（10 种） — `mp3` `aac` `m4a` `flac` `wav` `ogg` `aiff` `wma` `opus` `oga`

## 仓库管理

页面下方以卡片网格展示所有已连接的媒体仓库。

### 仓库卡片

每张卡片显示：

- **仓库名称** — 带有"主仓库"或"云仓库"标记（如适用）。
- **路径** — 仓库的文件系统路径或云端位置。
- **资源数量** — 该仓库中已索引的资源总数。
- **云导入状态**（仅限云仓库）— 最近一次云导入运行的状态、导入数量和失败数量。

### 仓库操作

每张卡片上的菜单按钮提供以下维护操作：

| 操作 | 说明 |
|------|------|
| **重新扫描** | 扫描仓库文件系统，检测新增、变更或删除的文件 |
| **检测堆叠** | 自动检测连拍、包围曝光等照片序列并归为堆叠，检测结果可在[资源库](/zh-cn/user-manual/features/assets#堆叠管理)中查看 |
| **扫描重复文件** | 执行精确匹配和感知哈希两种方式的重复检测 |
| **重建位置聚类** | 根据 EXIF 地理位置信息重建位置聚类 |
| **从云端导入**（仅云仓库） | 触发云仓库到本地的文件导入 |

每个操作独立追踪加载状态，执行期间对应按钮显示旋转动画。

### 新增仓库

单击"添加库"按钮打开弹窗，按以下步骤创建新仓库：

1. 输入仓库名称。
2. 选择来源类型：**本地** 或 **云端**。
3. 若选择云端，从下拉列表中选择已连接的[云凭据](/zh-cn/user-manual/features/settings#云同步)。
4. 单击"创建库"提交。

### 扫描全部

单击"扫描所有库"按钮可对所有仓库同时发起文件系统扫描。
