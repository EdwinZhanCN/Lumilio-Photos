## 概览

设置页面提供对应用各项配置的集中管理入口。页面以标签页形式组织，不同标签页涵盖账户、外观、服务器、AI、云同步和用户管理等范畴。

::: tip 提示
AI、云同步和用户三个标签页仅对管理员可见，普通用户不会看到这些入口。
:::

## 标签页结构

设置页面顶部为标签栏，通过单击可在各标签页间切换。标签页切换记录在浏览器地址栏的 `?tab=` 参数中，支持直接链接到特定标签页。

| 标签页 | 说明 | 可见范围 |
|--------|------|---------|
| **账户** | 个人资料、安全设置 | 所有用户 |
| **外观** | 界面语言、主题、布局偏好 | 所有用户 |
| **服务器** | 工作库选择、运行状态 | 所有用户 |
| **AI** | AI 代理与机器学习功能 | 仅管理员 |
| **云同步** | 云服务凭据管理 | 仅管理员 |
| **用户** | 用户账户管理 | 仅管理员 |

## 账户

账户标签页分为个人资料和安全两组。

### 个人资料

- **头像** — 从资源库中选择一张照片作为头像。
- **显示名称** — 自定义界面中显示的名称。
- **用户名** — 只读显示，格式为 `@用户名`。
- **角色** — 只读显示当前角色（管理员 / 用户）。

可点击"查看权限"按钮打开弹窗，查看当前账户拥有的所有权限及其启用状态。

### 安全

- **密码** — 点击跳转至修改密码页面。
- **验证器应用（TOTP）** — 显示是否已启用双重认证及剩余恢复码数量。点击可设置或关闭双重认证。
- **通行密钥** — 显示已注册的通行密钥数量。点击可管理 WebAuthn 通行密钥（新增或删除）。

::: warning 注意
通行密钥需要先启用 TOTP 双重认证后方可添加。
:::

## 外观

外观设置仅影响当前登录设备，修改后即时生效，无需手动保存。

::: tip 提示
外观偏好存储在浏览器本地，不同设备可独立设置。
:::

### 语言与地区

- **界面语言** — 选择应用界面的显示语言（English / 中文）。
- **地区** — 选择地区格式（中国 / 其他），影响地图服务商等区域化行为。

### 主题

- **跟随系统主题** — 开启后界面主题自动匹配操作系统的明暗模式。
- **浅色主题** — 从预设的浅色主题列表中选取。
- **深色主题** — 从预设的深色主题列表中选取。

每个主题以色块缩略图预览主色、辅助色和强调色。

### 资源库布局

- **完整布局** — 自适应瓦片排列的浏览布局。
- **紧凑布局** — 规整网格排列的浏览布局。
- **紧凑列数** — 拖动滑块调整网格模式下的列数（4–10 列）。

## 服务器

### 工作库

从下拉菜单中选择当前应用使用的工作库（或选择"所有工作库"）。此选择会影响资源库、首页、地图、统计、上传和 ML 索引等模块的数据范围。

### 健康检查间隔

拖动滑块或点击预设按钮（1s、2s、5s、10s、30s、50s），设置前端轮询服务器健康状态的间隔时间。

### 运行时信息

只读展示服务器的运行时配置，包括运行环境、服务端口、存储根目录、硬件加速状态、扫描策略、日志级别、地理编码服务商等。

## AI

此标签页仅对管理员可见。修改后需手动保存，支持先验证再保存。

### 代理配置

- **启用代理** — AI 助手（Lumilio Agent）的主开关。
- **服务商** — 选择 LLM 服务商（unset / ark / openai / deepseek / ollama）。
- **模型名称** — 填写使用的模型名称。
- **接口地址** — 填写服务商的 API 端点地址。
- **API 密钥** — 填写服务商的 API 密钥（输入后不会回显，仅标记是否已配置）。

### ML 功能

- **图像语义分析** — 启用后可基于图片语义内容进行搜索。
- **BioCLIP 物种识别** — 启用后可对照片中的物种进行分类识别。
- **OCR 文字识别** — 启用后可提取图像中的文字信息。
- **人物识别** — 启用后可检测和识别照片中的人脸。

## 云同步

此标签页仅对管理员可见。用于管理与第三方云存储服务的连接凭据。

- **添加凭据** — 从可用云服务商列表中选择，填写凭据信息（具体字段由各服务商定义）。
- **已添加的凭据** — 列表中显示每个凭据的名称、服务商、状态（已连接 / 待验证 / 错误 / 已禁用）。
- **操作** — 每个凭据可执行重连、断开连接或删除操作。部分服务商在添加时需完成额外的身份验证挑战。

## 用户

此标签页仅对管理员可见。用于管理所有用户账户。

用户列表显示每个用户的头像、显示名称、用户名、资源数量、相册数量、角色和活跃状态。

选中一个用户后可编辑：

- **头像** — 设置或更换用户的头像。
- **用户名** — 编辑用户名（有格式校验）。
- **显示名称** — 编辑显示名称。
- **角色** — 在普通用户和管理员之间切换。
- **状态** — 启用或停用账户。

### 重置访问

点击"重置访问"可生成临时密码，同时清除该用户的所有双重认证因素（TOTP + 通行密钥 + 恢复码）。生成的临时密码可复制到剪贴板。不可对自己的账户执行此操作。
