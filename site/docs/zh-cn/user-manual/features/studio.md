## 概览

工作室是一个非破坏性的照片编辑器，内置于资源库中。你可以对照片进行曝光、色彩、细节等调整，应用装饰性边框，所有编辑以侧边文件形式保存，原始文件永远不会被修改。

## 进入工作室

有三种方式进入工作室：

1. 从侧边栏单击「工作室」导航项进入主页，然后选择照片开始编辑。
2. 在[资源库](/zh-cn/user-manual/features/assets)的全屏查看器中，打开导出弹窗并单击画笔图标，将当前查看的资源发送到工作室。
3. 直接访问带 `?assetId=` 参数的链接（如 `/studio?assetId=xxx`）可跳过选择器直接进入编辑器。

## 工作流程

```
工作室主页 → 照片选择器 → 编辑器 → 保存 / 导出
```

### 主页

主页包含以下内容：

- **「打开编辑器」** — 进入照片选择器挑选要编辑的照片。
- **「添加边框」** — 进入照片选择器，选中后直接进入编辑器并展开边框工具。
- **最近编辑** — 显示最近编辑过的照片列表（最多 12 张），可单击恢复编辑或清空列表。

### 照片选择器

照片选择器以网格形式展示资源库中的照片，支持按类型筛选。选中一张照片后进入编辑器。

## 编辑器

编辑器采用三栏布局：

- **左侧** — 资产信息面板：缩略图、文件名、尺寸、MIME 类型、EXIF 数据。
- **中间** — 预览视口：支持缩放（Ctrl + 滚轮）、平移、适应屏幕，按住按钮可查看编辑前后的对比。
- **右侧** — 调整面板：包含开发调整和边框工具。

顶栏显示文件名、保存状态、渲染引擎标识，以及撤销、重置、保存和导出按钮。

### 开发调整

调整项分组为三个可折叠区域：

**光线**
| 调整 | 范围 | 说明 |
|------|------|------|
| 曝光 | -3 ~ +3 EV | 整体亮度调节 |
| 对比度 | -100 ~ +100 | 明暗对比强度 |
| 高光 | -100 ~ +100 | 亮部区域还原 |
| 阴影 | -100 ~ +100 | 暗部区域提亮 |
| 白色 | -100 ~ +100 | 白色色阶 |
| 黑色 | -100 ~ +100 | 黑色色阶 |

**色彩**
| 调整 | 范围 | 说明 |
|------|------|------|
| 色温 | -100 ~ +100 | 白平衡色温 |
| 色调 | -100 ~ +100 | 白平衡色调 |
| 鲜艳度 | -100 ~ +100 | 低饱和度区域的饱和度增强 |
| 饱和度 | -100 ~ +100 | 整体色彩饱和度 |

**细节**
| 调整 | 范围 | 说明 |
|------|------|------|
| 清晰度 | -100 ~ +100 | 局部对比度增强 |
| 锐度 | 0 ~ +100 | 边缘锐化 |
| 降噪 | 0 ~ +100 | 减少噪点 |

### 几何变换

- **向左旋转 / 向右旋转** — 每次 90 度
- **水平翻转 / 垂直翻转** — 镜像翻转

### 边框工具

边框工具提供 5 种装饰模式，应用后会在编辑后的照片上叠加视觉效果：

| 模式 | 说明 | 可调参数 |
|------|------|---------|
| **彩色** | 纯色相框 | 边框宽度、颜色 |
| **磨砂** | 模糊提亮的背景，中心图像缩小并带圆角 | 模糊强度、亮度、圆角半径 |
| **暗角** | 径向暗角效果 | 暗角强度 |
| **磨砂信息** | 磨砂效果 + 底部 EXIF 信息条（相机型号、拍摄参数） | 模糊强度、亮度、圆角半径 |
| **信息条** | 底部白色信息条，显示相机/镜头型号、品牌标志和拍摄参数 | 无（EXIF 为只读） |

> 边框工具会烘焙新像素，不会保存到侧边文件中。如需保留边框效果，请通过导出下载。

## 保存与导出

### 保存

单击顶栏「保存」按钮，当前的所有开发调整（光线、色彩、细节、几何变换）会被写入侧边文件，与原始文件分开存储。下次打开该照片编辑时，调整会自动恢复。

### 导出

| 当前状态 | 导出格式 |
|---------|---------|
| 仅开发调整（无边框） | JPEG，质量 92%，导出为 `{文件名}-lumilio.jpg` |
| 已应用边框 | PNG（无损），导出为 `{文件名}-border.png` |

> 工作室仅支持照片编辑，视频和音频资源不支持此功能。
