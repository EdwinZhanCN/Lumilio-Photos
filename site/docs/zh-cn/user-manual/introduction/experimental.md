# 实验性功能与已知风险

::: warning 注意
流明集当前处于 Beta 阶段。请先使用测试媒体或已有可靠备份的资料库进行试用，不要将本应用作为重要媒体的唯一存储位置。
:::

除项目整体仍处于 Beta 阶段外，流明集还包含少量不能提供长期兼容保证的实验性能力。这些能力会明确标记，且不作为基础媒体管理的必要条件。

## 如何理解“永久实验性”

**永久实验性**表示某项功能依赖非官方或不稳定的外部行为。即使流明集未来进入稳定版本，该功能仍可能在没有预告的情况下部分失效、完全失效或需要重新设计。

## iCloud 导入

::: danger 永久实验性功能
流明集的 iCloud 导入**不是 Apple 官方提供或支持的集成**。其实现参考开源项目 [chyroc/icloudgo](https://github.com/chyroc/icloudgo)，并依赖 Apple 当前的网络服务行为。
:::

Apple 可能随时调整认证流程、请求接口、返回字段或服务策略。这些变化可能导致：

- Apple ID 要求双重认证、重新验证，或既有会话提前过期。
- 登录或自动访问触发 Apple 的账号安全机制。
- 相册分页、元数据或媒体版本行为变化。
- 部分照片、视频或 Live Photo 组件下载失败或缺失。
- 功能在流明集未发布更新之前无法继续使用。

使用该功能可能触发 Apple 的账号安全机制。**用户需要自行评估并承担使用非官方集成的账号风险。**流明集不保证该功能长期可用，也无法代表 Apple 解释或解除账号限制。

## 使用前后请核验

1. 首先选择少量非关键媒体进行测试，不要立即导入整个资料库。
2. 导入后核对项目数量、文件大小、可读性与关键元数据。
3. 对 Live Photo 分别检查静态照片和视频组件。
4. 不要将“任务完成”直接等同于所有云端媒体均已完整导入。
5. 在确认本地副本和独立备份均可用前，不要删除 iCloud 中的原件。

流明集的导入流程不会主动删除 iCloud 中的原件。导入凭据与会话数据属于 app-private state，不写入媒体存储位置；应保护运行流明集的应用数据目录、密钥文件和数据库备份。

[云端导入](../features/cloud-import.md)功能页已经建立，但认证、重连、导入状态与故障排查步骤尚未完成。在这些步骤可以被完整验证前，不建议启用 iCloud 导入。

## 其他 Beta 限制

- 不同平台、媒体格式和外部工具的兼容性仍在持续验证。
- 大型资料库的首次扫描、缩略图生成和索引可能需要较长时间。
- AI 与机器学习结果可能出错，需要由用户复核。
- 某些功能可能依赖外部工具、模型、硬件或网络服务。

实验性功能失败时，流明集应尽可能保持本地仓库和基础媒体管理可用。如果你发现实验性功能影响了本地媒体或其他核心路径，请停止相关任务并报告问题。
